# Skip-Gram Model (Word2Vec) — Deep Intuition

Skip-Gram is one of the two architectures of **Word2Vec** introduced by
Tomas Mikolov at
Google.

The **core idea** is:

> Given a word, predict the surrounding words (context).

It learns **word embeddings** such that words used in similar contexts have similar vectors.

---

# 1️⃣ Intuition: How humans understand words

Imagine you see the word:

```
apple
```

You probably think of words like:

```
fruit
eat
tree
red
sweet
```

Why?

Because these words **frequently appear near “apple”**.

Skip-Gram uses this same intuition:

> If two words appear in similar contexts, they have similar meaning.

# 3️⃣ Why it is called "Skip-Gram"

The word **“skip”** means the model **skips over the center word to predict context words**.

Example

Sentence

```
I love machine learning
```

Center word

```
machine
```

Context words

```
love, learning
```

Training pairs

```
(machine → love)
(machine → learning)
```

---

# 4️⃣ Neural Network Structure

Skip-Gram uses a **very simple neural network**.

```
Input Word
     ↓
Embedding Layer (Hidden layer)
     ↓
Output Layer (Predict context words)
```

Layers:

| Layer  | Purpose                      |
| ------ | ---------------------------- |
| Input  | One-hot encoded word         |
| Hidden | Word embedding               |
| Output | Probability of context words |

---

# 5️⃣ Step-by-Step Example (Very Important)

Vocabulary:

```
["cat", "dog", "mat", "sits"]
```

Vocabulary size = **4**

---

### Step 1: One-Hot Encoding

Example word **cat**

```
cat → [1,0,0,0]
```

dog

```
dog → [0,1,0,0]
```

---

### Step 2: Input to Neural Network

Input word:

```
cat
```

Vector:

```
[1,0,0,0]
```

---

### Step 3: Embedding Layer

Suppose embedding size = **3**

Weight matrix:

```
4 × 3
```

Example matrix

```
cat → [0.2, 0.7, 0.1]
dog → [0.3, 0.8, 0.2]
mat → [0.1, 0.4, 0.9]
sits → [0.6, 0.2, 0.5]
```

When we multiply

```
OneHot × WeightMatrix
```

We get the **embedding vector**.

Example

```
cat → [0.2,0.7,0.1]
```

This is the **word embedding**.

---

# 6️⃣ Predicting Context Words

Now the network predicts probabilities of all words.

Example output:

| Word | Probability |
| ---- | ----------- |
| cat  | 0.02        |
| dog  | 0.10        |
| mat  | 0.30        |
| sits | 0.58        |

If true context word is **sits**, model adjusts weights.

Training uses **backpropagation**.

---

# 7️⃣ Why Skip-Gram Works

Words appearing in similar contexts get similar vectors.

Example sentences

```
The cat drinks milk
The dog drinks milk
```

Training pairs:

```
(cat → drinks)
(dog → drinks)
```

So embeddings learn:

```
cat ≈ dog
```

because they share context.

---

# 8️⃣ Word Similarity Learned

After training:

| Word  | Vector            |
| ----- | ----------------- |
| cat   | [0.25,0.10,0.80]  |
| dog   | [0.24,0.11,0.79]  |
| apple | [-0.70,0.33,0.12] |

Notice:

```
cat ≈ dog
```

but far from

```
apple
```

# 9️⃣ Python Example (Skip-Gram)

We use **Gensim**

Install

```python
pip install gensim
```

---

### Training Skip-Gram

```python
from gensim.models import Word2Vec

sentences = [
    ["the","cat","sits","on","the","mat"],
    ["the","dog","sits","on","the","rug"],
    ["cats","and","dogs","are","animals"]
]

model = Word2Vec(
    sentences,
    vector_size=100,
    window=2,
    min_count=1,
    sg=1   # sg=1 means Skip-Gram
)
```

---

### Get word vector

```python
print(model.wv["cat"])
```

Example output

```
[0.24, -0.11, 0.87, ...]
```

---

# 10️⃣ Finding Similar Words

```python
model.wv.most_similar("cat")
```

Output

```
[('dog', 0.88), ('kitten', 0.81)]
```

This means embeddings captured similarity.

---

# 11️⃣ Word Analogy Example

```python
model.wv.most_similar(
    positive=["king","woman"],
    negative=["man"]
)
```

Output

```
queen
```

Because

```
king - man + woman ≈ queen
```

# 12️⃣ Why Skip-Gram is Powerful

Compared to **CBOW**

| Feature                         | SkipGram |
| ------------------------------- | -------- |
| Works better with rare words    | ✔        |
| Captures semantic relations     | ✔        |
| Generates more training samples | ✔        |

CBOW is faster but Skip-Gram is usually **more accurate**.

# 14️⃣ Important Hyperparameters

| Parameter   | Meaning             |
| ----------- | ------------------- |
| vector_size | embedding dimension |
| window      | context window      |
| min_count   | ignore rare words   |
| sg          | 1 = SkipGram        |
| epochs      | training iterations |

Example

```python
model = Word2Vec(
    sentences,
    vector_size=300,
    window=5,
    min_count=2,
    sg=1,
    epochs=10
)
```

# 16️⃣ Key Intuition Summary

Skip-Gram learns embeddings using this idea:

```
Word → Predict surrounding words
```

If two words appear in **similar contexts**

```
Their vectors become similar
```

Example

```
cat → sits, eats, milk
dog → sits, eats, milk
```

So embeddings learn:

```
cat ≈ dog
```

---

✅ **One-line definition**

> Skip-Gram is a Word2Vec model that learns word embeddings by predicting surrounding context words from a center word.
