# Lemmatization

Lemmatization is a **very important text preprocessing technique in NLP**. It is similar to stemming but **more accurate and linguistically correct**.

Let’s go step-by-step with **clear intuition, examples, and Python using NLTK**.

---

# 1. What is Lemmatization?

**Lemmatization** reduces a word to its **base or dictionary form**, called the **lemma**.

Unlike stemming, it **uses vocabulary and grammar rules**.

Example:

| Word    | Lemma |
| ------- | ----- |
| running | run   |
| better  | good  |
| studies | study |
| cars    | car   |

Example sentence:

```
"I am running and he runs every day"
```

After lemmatization:

```
["I", "am", "run", "and", "he", "run", "every", "day"]
```

So:

```
running → run
runs → run
```

---

# 2. Why Lemmatization is Important

Without lemmatization:

```
run
running
runs
ran
```

Model treats them as **4 different words**.

With lemmatization:

```
run
```

Benefits:

* reduces vocabulary size
* keeps **valid words**
* better for **NLP tasks**
* improves model accuracy

---

# 3. Stemming vs Lemmatization

| Feature  | Stemming             | Lemmatization        |
| -------- | -------------------- | -------------------- |
| Approach | rule-based           | dictionary + grammar |
| Accuracy | lower                | higher               |
| Output   | may not be real word | always real word     |
| Example  | studies → studi      | studies → study      |

Example:

```
studies
```

Stemming:

```
studi
```

Lemmatization:

```
study
```

# 7. Basic Lemmatization Example

### Step 1 – Import library

```python
from nltk.stem import WordNetLemmatizer
```

### Step 2 – Create object

```python
lemmatizer = WordNetLemmatizer()
```

### Step 3 – Lemmatize words

```python
words = ["running", "runs", "ran", "cars", "studies"]

for word in words:
    print(word, "→", lemmatizer.lemmatize(word))
```

Output:

```
running → running
runs → run
ran → ran
cars → car
studies → study
```

Notice:

```
running → running
```

Why?

Because **WordNet needs the word's POS (Part of Speech)**.

---

# 8. Lemmatization with POS Tag (Important)

Words behave differently depending on **grammar role**.

Example:

```
running
```

Could be:

* noun
* verb

So we specify:

```
pos = 'v'
```

Example code:

```python
from nltk.stem import WordNetLemmatizer

lemmatizer = WordNetLemmatizer()

print(lemmatizer.lemmatize("running", pos="v"))
```

Output:

```
run
```

# 9. Lemmatizing a Sentence

Sentence:

```
"The players were running and playing matches"
```

### Python Example

```python
from nltk.tokenize import word_tokenize
from nltk.stem import WordNetLemmatizer

sentence = "The players were running and playing matches"

lemmatizer = WordNetLemmatizer()

words = word_tokenize(sentence)

lemmatized_words = []

for word in words:
    lemmatized_words.append(lemmatizer.lemmatize(word))

print(lemmatized_words)
```

Output:

```
['The', 'player', 'were', 'running', 'and', 'playing', 'match']
```

Notice verbs **didn't change properly**.

We must provide **POS tags**.

---

# 10. Lemmatization with POS Tagging (Best Practice)

First download POS tagger:

```python
nltk.download('averaged_perceptron_tagger')
```

### Code

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.stem import WordNetLemmatizer

sentence = "The players were running and playing matches"

lemmatizer = WordNetLemmatizer()

words = word_tokenize(sentence)

for word in words:
    print(word, "→", lemmatizer.lemmatize(word, pos="v"))
```

Output:

```
The → The
players → players
were → be
running → run
playing → play
matches → match
```

# 12. Practical NLP Example

Text:

```
"Players are playing matches and played yesterday"
```

### Code

```python
from nltk.tokenize import word_tokenize
from nltk.stem import WordNetLemmatizer

text = "Players are playing matches and played yesterday"

lemmatizer = WordNetLemmatizer()

words = word_tokenize(text)

lemmatized = [lemmatizer.lemmatize(word, pos='v') for word in words]

print(lemmatized)
```

Output:

```
['Players', 'be', 'play', 'match', 'and', 'play', 'yesterday']
```
