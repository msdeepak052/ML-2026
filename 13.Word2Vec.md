## Word2Vec (for NLP) — Detailed but Easy Explanation

### 1️⃣ What problem does Word2Vec solve?

In **Natural Language Processing**, computers cannot understand words directly.
So we convert words into numbers.

Earlier methods like **One-Hot Encoding** had problems:

| Word  | Vector      |
| ----- | ----------- |
| King  | [0,0,1,0,0] |
| Queen | [0,1,0,0,0] |
| Apple | [1,0,0,0,0] |

Problems ❌

* Vectors are **very large**
* **No semantic meaning**
* King and Queen look unrelated even though they are similar.

👉 **Word2Vec solves this** by learning **dense vectors (embeddings)** where similar words have similar vectors.

Example learned vectors:

| Word  | Vector                   |
| ----- | ------------------------ |
| King  | [0.25, 0.11, 0.90, ...]  |
| Queen | [0.24, 0.10, 0.88, ...]  |
| Apple | [-0.90, 0.33, 0.12, ...] |

Now **King and Queen vectors are close**.

---

# 2️⃣ What is Word2Vec?

**Word2Vec** is a neural network model developed by
Tomas Mikolov at
Google in 2013.

It converts **words → dense vector representations (embeddings).**

Goal:

> Words that appear in **similar contexts have similar meanings**

Example sentence

```
The cat sits on the mat
```

Word2Vec learns relationships like:

```
King - Man + Woman ≈ Queen
Paris - France + Italy ≈ Rome
```

---

# 3️⃣ Two Word2Vec Architectures

Word2Vec has **two models**

| Model         | Idea                                   |
| ------------- | -------------------------------------- |
| **CBOW**      | Predict word from surrounding context  |
| **Skip-Gram** | Predict context words from center word |

---

# 4️⃣ CBOW (Continuous Bag Of Words)

### Idea

Predict the **target word using surrounding words**

Example sentence

```
The cat sits on the mat
```

Window size = 2

For word **sits**

Context words:

```
cat, on
```

Training sample:

```
Input: [cat, on]
Output: sits
```

More examples:

| Context   | Target |
| --------- | ------ |
| The, sits | cat    |
| cat, on   | sits   |
| sits, the | on     |

---

### CBOW Neural Network Structure

```
Input Layer (context words)
        ↓
Hidden Layer (embedding)
        ↓
Output Layer (target word)
```

Important idea:

* Input = One-Hot vectors
* Hidden layer weights = **Word embeddings**

After training → Hidden layer weights become **word vectors**

---

### CBOW Example in Python

We usually use **Gensim**

Install

```python
pip install gensim
```

Example

```python
from gensim.models import Word2Vec

sentences = [
    ["the","cat","sits","on","the","mat"],
    ["the","dog","sits","on","the","rug"],
    ["cats","and","dogs","are","animals"]
]

model = Word2Vec(
    sentences,
    vector_size=50,
    window=2,
    min_count=1,
    sg=0   # 0 = CBOW
)

print(model.wv["cat"])
```

Output

```
[0.13, -0.22, 0.91, ...]
```

This is the **embedding vector for "cat"**

---

# 5️⃣ Skip-Gram Model

### Idea

Opposite of CBOW

Predict **context words from center word**

Example

Sentence

```
The cat sits on the mat
```

Center word = **sits**

Context words

```
cat, on
```

Training samples

| Input | Output |
| ----- | ------ |
| sits  | cat    |
| sits  | on     |

More examples

| Input | Output |
| ----- | ------ |
| cat   | the    |
| cat   | sits   |

---

### Skip-Gram Neural Network

```
Input word
   ↓
Embedding Layer
   ↓
Predict surrounding words
```

Skip-Gram works **better for rare words**

---

### Python Example (Skip-Gram)

```python
from gensim.models import Word2Vec

sentences = [
    ["the","cat","sits","on","the","mat"],
    ["the","dog","runs","in","the","park"]
]

model = Word2Vec(
    sentences,
    vector_size=100,
    window=2,
    min_count=1,
    sg=1   # 1 = SkipGram
)

vector = model.wv["cat"]
print(vector)
```

Sentence

Context for **machine**

```

```

# 7️⃣ Word Similarity

One of the coolest features.

Example

```python
model.wv.most_similar("cat")
```

Output

```
[('dog', 0.87), ('kitten', 0.81)]
```

This shows **semantic understanding**

---

# 8️⃣ Word Analogies

Word2Vec can solve analogies.

Example

```python
model.wv.most_similar(
    positive=["king","woman"],
    negative=["man"]
)
```

Output

```
queen
```

Because vectors learn relationships.

```
king - man + woman ≈ queen
```

# 9️⃣ How Word2Vec Training Actually Works

Steps

1️⃣ Convert words → one hot vectors
2️⃣ Pass through neural network
3️⃣ Learn weights using **backpropagation**
4️⃣ Hidden layer weights = **word embeddings**

Loss function uses **Softmax**

But computing Softmax over **large vocabulary is expensive**, so two optimizations are used:

| Method               | Idea                             |
| -------------------- | -------------------------------- |
| Negative Sampling    | Train only few negative examples |
| Hierarchical Softmax | Tree structure prediction        |

Example

```python
model.wv.similarity("cat","dog")
```

# 11️⃣ Visualizing Word Embeddings

We can use **PCA or t-SNE**

Example

```python
from sklearn.decomposition import PCA
import matplotlib.pyplot as plt

words = ["cat","dog","king","queen","apple"]
vectors = [model.wv[w] for w in words]

pca = PCA(n_components=2)
result = pca.fit_transform(vectors)

for i, word in enumerate(words):
    plt.scatter(result[i,0], result[i,1])
    plt.text(result[i,0], result[i,1], word)

plt.show()
```

This will show **similar words close together**

# 12️⃣ Advantages of Word2Vec

✅ Captures semantic meaning
✅ Lower dimensional vectors
✅ Efficient training
✅ Works well with large text

---

# 13️⃣ Limitations

❌ Context independent
Example

```
bank (river)
bank (money)
```

Both get same vector.

This problem was later solved by models like:

* BERT
* ELMo
* GPT

# 15️⃣ Quick Interview Summary

**Word2Vec**

• Converts words into dense vectors
• Based on **distributional hypothesis**

Two architectures:

1️⃣ **CBOW** – Predict word from context
2️⃣ **Skip-Gram** – Predict context from word

Key parameters

```
vector_size
window
min_count
sg
```

---

✅ **Simple one-line definition**

> Word2Vec is a neural network technique that learns vector representations of words such that words appearing in similar contexts have similar embeddings.
