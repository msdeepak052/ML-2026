# Stopwords

Stopwords are one of the **basic and very important steps in text preprocessing for NLP**. They help remove **common words that usually do not add meaningful information** for machine learning models.

Let’s understand this **step-by-step with intuition, examples, and Python using NLTK**.

---

# 1. What are Stopwords?

**Stopwords** are commonly used words in a language that usually **do not contribute much meaning to the text**.

Examples:

| Stopword | Meaning     |
| -------- | ----------- |
| the      | article     |
| is       | verb        |
| in       | preposition |
| and      | conjunction |
| a        | article     |
| to       | preposition |

Example sentence:

```text
"I am learning NLP and it is very interesting"
```

Tokens:

```text
["I","am","learning","NLP","and","it","is","very","interesting"]
```

After removing stopwords:

```text
["learning","NLP","interesting"]
```

---

# 2. Why Remove Stopwords?

Stopwords are removed because they:

1. **Appear very frequently**
2. **Do not carry important meaning**
3. **Increase dataset size**
4. **Reduce model efficiency**

Example:

Sentence:

```text
"The movie is very good and the acting is amazing"
```

Important words:

```text
movie
good
acting
amazing
```

Stopwords:

```text
the
is
and
very
```

Removing stopwords helps the model **focus on meaningful words**.

---

# 3. Stopwords in NLP Pipeline

Typical preprocessing pipeline:

```
Raw Text
↓
Lowercase
↓
Tokenization
↓
Stopwords Removal
↓
Stemming / Lemmatization
↓
Vectorization
```

Example:

Input:

```text
"The players are playing football in the stadium"
```

After stopword removal:

```text
["players","playing","football","stadium"]
```

# 7. Removing Stopwords from a Sentence

Sentence:

```text
"I am learning natural language processing and it is very interesting"
```

### Step 1 — Tokenization

### Step 2 — Remove stopwords

### Python Code

```python
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

sentence = "I am learning natural language processing and it is very interesting"

stop_words = set(stopwords.words('english'))

words = word_tokenize(sentence)

filtered_words = []

for word in words:
    if word.lower() not in stop_words:
        filtered_words.append(word)

print(filtered_words)
```

### Output

```text
['learning', 'natural', 'language', 'processing', 'interesting']
```

---

# 8. Using List Comprehension (Cleaner Code)

```python
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

sentence = "I am learning natural language processing and it is very interesting"

stop_words = set(stopwords.words('english'))

words = word_tokenize(sentence)

filtered_words = [word for word in words if word.lower() not in stop_words]

print(filtered_words)
```

Output:

```text
['learning', 'natural', 'language', 'processing', 'interesting']
```

# 9. Stopwords Removal Example (Full Text)

Text:

```text
"NLP is one of the most interesting fields in artificial intelligence"
```

### Code

```python
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

text = "NLP is one of the most interesting fields in artificial intelligence"

stop_words = set(stopwords.words('english'))

words = word_tokenize(text)

filtered = [word for word in words if word.lower() not in stop_words]

print(filtered)
```

Output:

```text
['NLP', 'one', 'interesting', 'fields', 'artificial', 'intelligence']
```

# 11. Custom Stopwords

Sometimes we add our **own stopwords**.

Example:

```python
custom_stopwords = {"hello","hi","thanks"}
```

### Example

```python
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

text = "hello I am learning NLP thanks"

stop_words = set(stopwords.words('english'))

custom = {"hello","thanks"}

stop_words = stop_words.union(custom)

words = word_tokenize(text)

filtered = [word for word in words if word.lower() not in stop_words]

print(filtered)
```

Output:

```text
['learning', 'NLP']
```

# 12. When Stopwords Should NOT Be Removed

Stopwords are **not always removed**.

Example tasks:

### Sentiment Analysis

Sentence:

```text
"I do not like this movie"
```

Stopword removal:

```text
['like','movie']
```

This **loses meaning**.

The word **"not" is critical**.

So sometimes we **keep stopwords**.
