# Part-of-Speech (POS) Tagging

Part-of-Speech (POS) Tagging is an important **text preprocessing technique in NLP**. It helps identify the **grammatical role of each word in a sentence**.

We'll go step-by-step with **intuitive explanation, examples, and Python code using NLTK**.

---

# 1. What is Part-of-Speech (POS) Tagging?

**POS tagging** is the process of labeling each word in a sentence with its **grammatical category**.

Example categories:

| POS              | Meaning              |
| ---------------- | -------------------- |
| Noun (NN)        | person, place, thing |
| Verb (VB)        | action               |
| Adjective (JJ)   | describes noun       |
| Adverb (RB)      | describes verb       |
| Pronoun (PRP)    | replaces noun        |
| Preposition (IN) | relation word        |

Example sentence:

```text
"The cat is sleeping peacefully"
```

POS tagging result:

```text
The        → DT   (Determiner)
cat        → NN   (Noun)
is         → VBZ  (Verb)
sleeping   → VBG  (Verb)
peacefully → RB   (Adverb)
```

---

# 2. Why POS Tagging is Important

POS tagging helps NLP systems understand **sentence structure and meaning**.

Example:

```text
book
```

Can be:

```text
Noun → "I read a book"
Verb → "I will book a ticket"
```

POS tagging helps the model **identify the correct meaning**.

Applications:

* Lemmatization
* Named Entity Recognition
* Machine Translation
* Question Answering
* Grammar checking

# 5. Basic POS Tagging Example

### Step 1 — Tokenize the sentence

### Step 2 — Apply POS tagging

### Python Code

```python
import nltk
from nltk.tokenize import word_tokenize

sentence = "NLP is very interesting"

words = word_tokenize(sentence)

pos_tags = nltk.pos_tag(words)

print(pos_tags)
```

Output:

```
[('NLP', 'NNP'),
 ('is', 'VBZ'),
 ('very', 'RB'),
 ('interesting', 'JJ')]
```

Explanation:

| Word        | POS         |
| ----------- | ----------- |
| NLP         | Proper noun |
| is          | Verb        |
| very        | Adverb      |
| interesting | Adjective   |

# 7. POS Tagging a Paragraph

Text:

```
"Machine learning is transforming the world."
```

### Python Code

```python
from nltk.tokenize import word_tokenize
import nltk

text = "Machine learning is transforming the world."

words = word_tokenize(text)

pos_tags = nltk.pos_tag(words)

for word, tag in pos_tags:
    print(word, ":", tag)
```

Output:

```
Machine : NN
learning : NN
is : VBZ
transforming : VBG
the : DT
world : NN
```

# 8. POS Tagging Multiple Sentences

Example text:

```
"NLP is amazing. It helps computers understand language."
```

### Python Code

```python
from nltk.tokenize import sent_tokenize, word_tokenize
import nltk

text = "NLP is amazing. It helps computers understand language."

sentences = sent_tokenize(text)

for sentence in sentences:
    words = word_tokenize(sentence)
    tags = nltk.pos_tag(words)
    print(tags)
```

Output:

```
[('NLP','NNP'),('is','VBZ'),('amazing','JJ'),('.','.')]
[('It','PRP'),('helps','VBZ'),('computers','NNS'),('understand','VB'),('language','NN'),('.','.')]
```

# 9. Common POS Tags in NLTK

| Tag | Meaning                |
| --- | ---------------------- |
| NN  | Noun                   |
| NNS | Plural noun            |
| NNP | Proper noun            |
| VB  | Verb                   |
| VBD | Past tense verb        |
| VBG | Verb ending with "ing" |
| JJ  | Adjective              |
| RB  | Adverb                 |
| PRP | Pronoun                |
| DT  | Determiner             |

Example:

```
playing → VBG
played → VBD
play → VB
```
