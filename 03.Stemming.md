# Stemming 
Stemming is one of the **core text preprocessing steps in NLP**. It helps reduce different forms of a word to a **common base form**, which improves model learning and reduces vocabulary size.

We’ll go **step-by-step with intuition, examples, and Python using NLTK**.

---

# 1. What is Stemming?

**Stemming** is the process of reducing words to their **root/base form (stem)** by removing suffixes.

Example:

| Original Word | Stem |
| ------------- | ---- |
| playing       | play |
| played        | play |
| player        | play |
| playing       | play |

Example sentence:

```
"I am playing football and he played yesterday"
```

After stemming:

```
["I", "am", "play", "football", "and", "he", "play", "yesterday"]
```

The idea is:

```
playing
played
player
plays
   ↓
play
```

This reduces **vocabulary size**.

---

# 2. Why Stemming is Important

In NLP models:

Without stemming:

```
play
playing
played
player
```

Model treats them as **4 different words**.

With stemming:

```
play
```

Now the model understands they belong to the **same concept**.

Benefits:

* reduces vocabulary
* improves search engines
* improves ML performance
* faster training

---

# 3. Stemming Process in NLP Pipeline

Typical preprocessing pipeline:

```
Raw Text
   ↓
Tokenization
   ↓
Lowercase
   ↓
Stopword Removal
   ↓
Stemming
   ↓
Vectorization
```

Example:

```
"I was playing football and players were happy"
```

After preprocessing:

```
["play","footbal","player","happi"]
```

Notice:

```
football → footbal
happy → happi
```

Stemming **does not guarantee real words**.

# 5. Porter Stemmer (Most Popular)

### Step 1 — Import Library

```python
from nltk.stem import PorterStemmer
```

---

### Step 2 — Create Stemmer Object

```python
stemmer = PorterStemmer()
```

---

### Step 3 — Stem Words

Example words:

```
playing
played
player
plays
```

### Python Code

```python
from nltk.stem import PorterStemmer

stemmer = PorterStemmer()

words = ["playing","played","player","plays"]

for word in words:
    print(word, "→", stemmer.stem(word))
```

### Output

```
playing → play
played → play
player → player
plays → play
```

Notice:

```
player → player
```

Because **Porter algorithm doesn't remove 'er' always**.

---

# 6. Stemming a Sentence

Now we combine **tokenization + stemming**.

Sentence:

```
"I love playing football and learning NLP"
```

### Step 1 — Tokenize

### Step 2 — Stem each word

### Python Code

```python
from nltk.tokenize import word_tokenize
from nltk.stem import PorterStemmer

sentence = "I love playing football and learning NLP"

words = word_tokenize(sentence)

stemmer = PorterStemmer()

stemmed_words = []

for word in words:
    stemmed_words.append(stemmer.stem(word))

print(stemmed_words)
```

### Output

```
['I', 'love', 'play', 'footbal', 'and', 'learn', 'NLP']
```

Notice:

```
football → footbal
learning → learn
```

---

# 7. Stemming Multiple Sentences

Example text:

```
"Players are playing football. They played yesterday."
```

### Python Code

```python
from nltk.tokenize import sent_tokenize, word_tokenize
from nltk.stem import PorterStemmer

text = "Players are playing football. They played yesterday."

stemmer = PorterStemmer()

sentences = sent_tokenize(text)

for sentence in sentences:
    
    words = word_tokenize(sentence)
    
    stemmed = [stemmer.stem(word) for word in words]
    
    print(stemmed)
```

### Output

```
['player', 'are', 'play', 'footbal', '.']
['they', 'play', 'yesterday', '.']
```

# 8. Lancaster Stemmer (More Aggressive)

Lancaster is **more aggressive** than Porter.

### Example Code

```python
from nltk.stem import LancasterStemmer

stemmer = LancasterStemmer()

words = ["playing","played","player"]

for word in words:
    print(word, "→", stemmer.stem(word))
```

Output

```
playing → play
played → play
player → play
```

Notice:

```
player → play
```

Lancaster removes **more suffixes**.

---

# 9. Snowball Stemmer

Snowball is an **improved version of Porter**.

### Code

```python
from nltk.stem import SnowballStemmer

stemmer = SnowballStemmer("english")

words = ["running","runs","ran"]

for word in words:
    print(word,"→",stemmer.stem(word))
```

Output

```
running → run
runs → run
ran → ran
```

# 12. Limitations of Stemming

Problems:

1. May produce **non-real words**

```
studies → studi
happy → happi
```

2. Sometimes **incorrect meaning**

```
universe → univers
```

Because stemming uses **rule-based suffix removal**.
