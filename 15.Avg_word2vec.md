# Average Word2Vec — Detailed Intuition (with Examples)

When we learn **Word2Vec**, we get **vector representations for individual words**.

But many NLP tasks require **sentence vectors** or **document vectors**.

Example tasks:

* Sentiment analysis
* Document classification
* Search relevance
* Text similarity

To solve this, one simple technique is **Average Word2Vec**.

---

# 1️⃣ The Core Idea

Average Word2Vec means:

> Represent a sentence by averaging the vectors of all its words.

Formula intuition:

Sentence vector =

```text
(word1_vector + word2_vector + ... + wordN_vector) / N
```

So the sentence meaning becomes the **average meaning of its words**.

---

# 2️⃣ Simple Example

Sentence:

```text
I love machine learning
```

Suppose Word2Vec gives these embeddings (dimension = 3).

| Word     | Vector          |
| -------- | --------------- |
| I        | [0.1, 0.3, 0.5] |
| love     | [0.7, 0.8, 0.9] |
| machine  | [0.4, 0.2, 0.1] |
| learning | [0.6, 0.5, 0.4] |

---

### Step 1 — Sum vectors

```
[0.1+0.7+0.4+0.6,
 0.3+0.8+0.2+0.5,
 0.5+0.9+0.1+0.4]
```

Result:

```
[1.8, 1.8, 1.9]
```

---

### Step 2 — Divide by number of words (4)

```
[1.8/4, 1.8/4, 1.9/4]
```

Final sentence vector:

```
[0.45, 0.45, 0.475]
```

This becomes the **sentence representation**.

---

# 3️⃣ Why Averaging Works (Intuition)

Word2Vec captures **semantic meaning**.

Example:

| Word  | Meaning |
| ----- | ------- |
| cat   | animal  |
| dog   | animal  |
| apple | fruit   |

Sentence:

```text
cat eats food
```

Vectors:

```
cat → [animal]
eats → [action]
food → [object]
```

Average vector represents:

```
animal + action + object
```

So the sentence meaning is captured.

# 5️⃣ Python Example (Step-by-Step)

First train Word2Vec.

```python
from gensim.models import Word2Vec

sentences = [
    ["i","love","machine","learning"],
    ["machine","learning","is","powerful"],
    ["deep","learning","is","cool"]
]

model = Word2Vec(
    sentences,
    vector_size=100,
    window=3,
    min_count=1
)
```

---

# 6️⃣ Get Word Vectors

Example:

```python
print(model.wv["machine"])
```

Output (example)

```
[0.14, -0.22, 0.91, ...]
```

---

# 7️⃣ Average Word2Vec for Sentence

Sentence:

```
"I love machine learning"
```

Convert into tokens.

```python
sentence = ["i","love","machine","learning"]
```

---

### Compute average

```python
import numpy as np

def average_word2vec(words, model):
    vectors = []

    for word in words:
        if word in model.wv:
            vectors.append(model.wv[word])

    return np.mean(vectors, axis=0)

sentence_vector = average_word2vec(sentence, model)

print(sentence_vector)
```

Output

```
[0.12, -0.09, 0.22, ...]
```

This is the **sentence embedding**.

---

# 8️⃣ Example with Multiple Sentences

Dataset:

```
"I love NLP"
"I enjoy machine learning"
"I hate bugs"
```

Compute sentence vectors.

```python
sentences = [
    ["i","love","nlp"],
    ["i","enjoy","machine","learning"],
    ["i","hate","bugs"]
]

sentence_vectors = []

for sentence in sentences:
    vec = average_word2vec(sentence, model)
    sentence_vectors.append(vec)
```

Now each sentence becomes a **numeric vector**.

---

# 9️⃣ Sentence Similarity Example

We can compare sentences using **cosine similarity**.

Example sentences:

```
I love NLP
I enjoy machine learning
```

Python code:

```python
from sklearn.metrics.pairwise import cosine_similarity

v1 = average_word2vec(["i","love","nlp"], model)
v2 = average_word2vec(["i","enjoy","machine","learning"], model)

similarity = cosine_similarity([v1], [v2])

print(similarity)
```

Output:

```
0.78
```

High similarity means sentences are related.

# 11️⃣ Advantages of Average Word2Vec

✅ Very simple
✅ Works well with small datasets
✅ Fast computation
✅ Easy to implement

No complex neural networks required.

---

# 12️⃣ Limitations

Average Word2Vec ignores **word order**.

Example:

```
dog bites man
man bites dog
```

Average vectors become almost identical.

Also ignores **context**.

Example:

```
bank (river)
bank (finance)
```

Both share the same embedding.

Modern models like:

* BERT
* GPT

solve these issues using **contextual embeddings**.

---

# 15️⃣ Interview Definition

**Average Word2Vec**

> Average Word2Vec represents a sentence or document by computing the average of the Word2Vec embeddings of all words in that text.

Formula:

```
Sentence_Vector = (sum of word vectors) / number of words
```

---

# 🔑 Key Takeaway

Word2Vec gives **word vectors**

Average Word2Vec converts them into **sentence vectors**

```
Words → Word2Vec → Word Embeddings
Words → Average → Sentence Embedding
```
